# Urunan

Create funding (urunan) to help people

join funding, help each other and get precious token !! 

## Installation

Use npm to install  [npm](https://www.npmjs.com/).

```shell
npm i
```

## Usage
### compile
```shell
npx hardhat compile
```

### Testing
```shell
npx hardhat test
```

### Another command 

```shell
npx hardhat help
REPORT_GAS=true npx hardhat test
npx hardhat node
npx hardhat run scripts/deploy.ts
```
